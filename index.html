    Text Hierarchy

    Primary text uses Ink at full opacity. Body copy, headings, and content where readability is critical.

    Secondary text uses Graphite. Descriptions, supporting information, and labels that recede behind primary.

    Tertiary text uses Slate. Captions, timestamps, footnotes, and metadata.

    Accent text uses the brand primary for links, interactive labels, and calls to action.

    Ink: #1A1625
    Graphite: #3D3551
    Slate: #6E6584
    Purple: #6B2FA0
